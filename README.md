# Ball215Lab1
What is the difference between frontend and backend responsibilities?
The difference between frontend and backend is the responsibilities of frontend involves the users end of the application while the backend manages the functionality on the server side.

Describe the request–response lifecycle.
The request-response lifecycle describes the communication between a client application and a server through data interactions starting from users actions, the http request sent to the database and server, and the response being returned to the front end.

Why is modular file organization important?
Modular file organization is important for keeping code organized and clean. Using the organization technique to break down code into modules makes maintenance and reusability easier.
